# Logo Files

Please place your logo files here:

- `logo.png` - Main logo (transparent background recommended)
- `logo-dark.png` - Optional: Logo for dark mode

The logo should be:
- PNG format with transparent background (recommended)
- Minimum 200x200px for best quality
- Optimized for web use

The code will automatically use `/images/logo.png` and fall back to text if the image is not found.
